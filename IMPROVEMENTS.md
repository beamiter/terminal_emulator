# Terminal Emulator - 改进日志

## v0.3.0 改进内容

### 1. **PTY 支持（基础）**
- 添加了 Unix/Linux PTY 框架（`src/pty.rs`）
- 支持 PTY 大小调整
- 准备就绪用于 Shell 集成

### 2. **增强的 UI**
- ✨ **光标闪烁效果** - 500ms 间隔闪烁
- 🎨 **改进的光标渲染** - 半透明块式光标，增强视觉效果  
- ✏️ **下划线支持** - 显示带有下划线的文本
- 🖱️ **更好的文本选择** - 改进的选择高亮显示
- 📊 **更完整的清屏支持** - J、K 命令的完整实现

### 3. **剪贴板和快捷键**
- ✂️ **Ctrl+V 粘贴** - 从系统剪贴板粘贴文本
- 🔍 **快捷键支持**：
  - `Ctrl+Up` / `Ctrl+Down` - 滚动终端
  - `Ctrl+C` / `Ctrl+D` - 系统信号（已存在）
  - `Tab` - 制表符

### 4. **ANSI 序列兼容性提升**
增加了以下转义序列支持：
- **光标控制**：
  - `ESC[E` - 光标下移到行首
  - `ESC[F` - 光标上移到行首  
  - `ESC[G` - 光标移至指定列
  - `ESC[S` - 向上滚动
  - `ESC[T` - 向下滚动

- **清屏命令**：
  - `ESC[0J` - 清屏到光标末尾
  - `ESC[1J` - 清屏从光标开始
  - `ESC[2J` - 清整个屏幕
  - 行清除支持（0/1/2 参数）

- **扩展颜色支持**：
  - `ESC[38;5;n` - 256 色索引模式
  - `ESC[38;2;r;g;b` - 真彩色 RGB 模式
  - `ESC[48;5;n` 和 `ESC[48;2;r;g;b` - 背景色

### 5. **代码结构优化**
- 模块化设计：
  - `pty.rs` - PTY 管理
  - `clipboard.rs` - 剪贴板操作
  - `terminal.rs` - 终端核心逻辑
  - `ui.rs` - UI 渲染
  - `color.rs` - 颜色处理

### 技术细节

#### 依赖更新
```toml
tokio = { version = "1.36", features = ["full"] }
parking_lot = "0.12"  # 改进的 Mutex 实现
```

#### 光标闪烁实现
500ms 间隔切换 `cursor_visible` 标志，提供真正的光标闪烁效果。

#### 扩展 SGR 参数
支持带中间参数的 SGR：
- `38;5;n` - 第 1 个参数是 38，第 2 个是 5，第 3 个是颜色索引
- `38;2;r;g;b` - 第 1 个参数是 38，第 2 个是 2，后面跟 RGB 值
